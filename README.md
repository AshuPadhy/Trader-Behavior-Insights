# Trader-Behavior-Insights

This repository contains the analysis scaffold for the "Trader Behavior vs Market Sentiment" assignment. It includes notebooks, scripts, and instructions to reproduce the analysis once the two input datasets are placed in data/raw/.

Structure
- data/
  - raw/  (place the two datasets here: hyperliquid_history.csv, fear_greed.csv)
  - processed/
- notebooks/
  - 01_data_prep.ipynb (template)
  - 02_exploratory_analysis.ipynb (template)
  - 03_modeling_and_tests.ipynb (template)
- src/
  - data_ingest.py
  - features.py
  - analysis_utils.py
- reports/
  - report.md (short report placeholder)
  - slides.md (slide deck placeholder)
- requirements.txt
- .gitignore

See data/README.md for where to place input files and how to run the notebooks.
